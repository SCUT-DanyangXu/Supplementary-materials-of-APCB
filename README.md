# Supplementary Material

This repository provides the supplementary material for the paper:

**Adaptive Partition-and-Combination Box for Power Flexibility Aggregation in Distribution Systems**

## Contents

### 1. Complete Distribution System Model

The supplementary document presents the complete mathematical formulation of the distribution system used to construct the exact aggregation model, including:

- Linearized DistFlow constraints;
- Nodal active- and reactive-power balance constraints;
- Voltage-drop and voltage-limit constraints;
- Branch apparent-power limits;
- Dispatchable generator constraints;
- Renewable energy source constraints;
- Energy storage system constraints;
- Substation power exchange constraints; and
- The definition of the projected multi-period substation active-power trajectory.




### 2. Modified IEEE 33-Bus System Data

The repository also contains the parameter data of the modified IEEE 33-bus distribution system used in the case studies, including:

- Network topology and branch parameters;
- Nodal load data;
- Dispatchable generator parameters;
- Renewable energy source profiles and parameters;
- Energy storage system parameters; and
- Other parameters required to reproduce the numerical experiments.

## Purpose

These supplementary materials are provided to facilitate the reproduction and verification of the numerical results reported in the paper.
